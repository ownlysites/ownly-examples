# Ownly Examples

Eight live one-page sample sites built by Ownly Web Studio.

| Route | Industry | Brand |
|---|---|---|
| `/hotel` | Coastal Boutique Hotel | La Marée |
| `/restaurant` | Modern American Bistro | Maker & Hearth |
| `/hvac` | Emergency HVAC | Tidewater HVAC |
| `/medspa` | Aesthetic Medicine | Luminé |
| `/wealth` | Private Wealth Advisory | Meridian Wealth |
| `/realestate` | Luxury Real Estate | The Hartley Group |
| `/saas` | Ops Analytics | Quanta |
| `/ai-agent` | Inbound AI Agent | Replywise |

All eight built with Next.js 15 + Tailwind 3, no external image dependencies — pure CSS + SVG + gradient compositions for offline parity.

Live: https://ownly-examples.vercel.app/

Built by [Ownly Web Studio](https://ownly-web-studio.vercel.app/).
